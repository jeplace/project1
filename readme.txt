Hello! This website was designed to showcase my favorite video game series, Fallout. It is stylized to follow the classic Vault-Tec color scheme from the games and utilize font choices to give off that retro vibe.

It features a simple navigation bar in the top left as well as using image links for the games and an audio player for the intro narration of the first game.

The Vault Boy icon in the top left links directly to the Bethesda Fallout webpage

Note: The Vault Boy and game links play a fun sound effect using javascript! (I know its not required but I really wanted to add this detail)
